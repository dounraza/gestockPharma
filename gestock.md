# 🚀 SaaS Inventory & Facturation

Application complète de gestion de stock, produits et facturation avec architecture SaaS.

---

# 🧠 Stack technique

## Frontend
- React (Vite)
- TailwindCSS
- Axios
- React Router

## Backend
- Node.js
- Express.js

## Backend as a Service
- Supabase (Auth + Database)
